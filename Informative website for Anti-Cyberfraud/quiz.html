Malware Identification Quiz

1. Which type of malware disguises itself as a legitimate program?

  • Trojan
  • Worm
  • Adware
Correct Answer: Trojan

2. Which type of malware spreads by sending copies of itself to other machines through a network?

  • Virus
  • Spyware
  • Ransomware
Correct Answer: Virus

3. Which type of malware encrypts files and demands payment for their release?

  • Worm
  • Ransomware
  • Rootkit
Correct Answer: Ransomware

4. Which type of malware records keystrokes to steal sensitive information such as passwords and credit card numbers?

  • Rootkit
  • Spyware
  • Adware
Correct Answer: Spyware

5. Which type of malware displays unwanted advertisements on a user's computer?

  • Rootkit
  • Adware
  • Worm
Correct Answer: Adware

6. Which type of malware launches a Distributed Denial of Service (DDoS) attack?

  • Trojan
  • Worm
  • Botnet
Correct Answer: Botnet

7. Which type of malware disguises itself as a harmless program but performs malicious activities in the background?

  • Rootkit
  • Spyware
  • Trojan
Correct Answer: Trojan

8. Which type of malware takes advantage of vulnerabilities in a computer system to gain unauthorized access?

  • Worm
  • Rootkit
  • Exploit Kit
Correct Answer: Exploit Kit

9. Which type of malware spreads by attaching itself to legitimate executable files?

  • Virus
  • Ransomware
  • Spyware
Correct Answer: Virus

10. Which type of malware monitors a user's web browsing habits and displays targeted advertisements?

  • Adware
  • Rootkit
  • Botnet
Correct Answer: Adware
