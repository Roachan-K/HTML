Cybercrime Quiz 4

1. Which of the following is NOT a fundamental principle of cryptography?

  • Confidentiality
  • Integrity
  • Scalability
Correct Answer: Scalability

2. What type of attack involves an attacker exploiting a flaw in a web application's authentication mechanism to gain unauthorized access?

  • Cross-Site Scripting (XSS)
  • Cross-Site Request Forgery (CSRF)
  • Session Hijacking
Correct Answer: Session Hijacking

3. Which cybersecurity attack involves an attacker intercepting and modifying communication between two parties without their knowledge?

  • Man-in-the-Middle (MitM)
  • Distributed Denial of Service (DDoS)
  • Ransomware
Correct Answer: Man-in-the-Middle (MitM)

4. What is the term used to describe a technique that involves an attacker guessing weak passwords by systematically checking all possible combinations?

  • Brute Force Attack
  • Phishing
  • Spoofing
Correct Answer: Brute Force Attack

5. Which encryption algorithm is considered the most secure and widely used for securing sensitive data?

  • DES (Data Encryption Standard)
  • RSA (Rivest-Shamir-Adleman)
  • AES (Advanced Encryption Standard)
Correct Answer: AES (Advanced Encryption Standard)

6. What is the term used to describe a security vulnerability that allows an attacker to execute malicious code by injecting SQL commands into input fields?

  • Cross-Site Scripting (XSS)
  • SQL Injection
  • Buffer Overflow
Correct Answer: SQL Injection

7. Which of the following is a cybersecurity principle that focuses on ensuring that only authorized users can access resources?

  • Availability
  • Confidentiality
  • Authentication
Correct Answer: Authentication

8. What is the term used to describe a security incident response plan that outlines the steps to be taken in the event of a cybersecurity breach?

  • Incident Recovery Plan
  • Disaster Recovery Plan
  • Cybersecurity Incident Response Plan
Correct Answer: Cybersecurity Incident Response Plan

9. Which cybersecurity attack involves an attacker masquerading as a legitimate entity in electronic communication?

  • Spoofing
  • Pharming
  • Worm
Correct Answer: Spoofing

10. Which cybersecurity concept involves intentionally luring attackers to a decoy system to monitor and analyze their behavior?

  • Honeypot
  • Sandbox
  • Firewall
Correct Answer: Honeypot
