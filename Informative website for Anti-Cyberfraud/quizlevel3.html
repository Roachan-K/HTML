Cybercrime Quiz 3

1. What is the term used to describe a cyberattack that occurs at the application layer and targets vulnerabilities in web applications?

  • XSS (Cross-Site Scripting)
  • SQL Injection
  • Buffer Overflow
Correct Answer: XSS (Cross-Site Scripting)

2. Which cryptographic algorithm is commonly used for secure communication over the Internet, providing encryption, authentication, and data integrity?

  • DES (Data Encryption Standard)
  • AES (Advanced Encryption Standard)
  • RSA (Rivest-Shamir-Adleman)
Correct Answer: AES (Advanced Encryption Standard)

3. What type of attack involves an attacker intercepting and modifying communication between two parties without their knowledge?

  • Man-in-the-Middle (MitM)
  • DDoS (Distributed Denial of Service)
  • SQL Injection
Correct Answer: Man-in-the-Middle (MitM)

4. What is the term used to describe a security mechanism that protects networks from unauthorized access by analyzing and filtering incoming and outgoing traffic?

  • IDS (Intrusion Detection System)
  • IPS (Intrusion Prevention System)
  • Firewall
Correct Answer: Firewall

5. Which of the following is a technique used by attackers to trick users into providing sensitive information by posing as a trustworthy entity?

  • Spear Phishing
  • Whaling
  • Vishing
Correct Answer: Spear Phishing

6. Which cybersecurity framework provides guidelines and best practices for organizations to manage and mitigate cybersecurity risks?

  • COBIT (Control Objectives for Information and Related Technologies)
  • ISO/IEC 27001
  • NIST Cybersecurity Framework
Correct Answer: NIST Cybersecurity Framework

7. Which term refers to the practice of disguising information to make it unreadable without the proper decryption key?

  • Hashing
  • Salting
  • Encryption
Correct Answer: Encryption

8. What is the term used to describe a software vulnerability that allows attackers to execute arbitrary code on a targeted system?

  • Logic Bomb
  • Zero-Day Exploit
  • Race Condition
Correct Answer: Zero-Day Exploit

9. What is the term used to describe a cyberattack in which an attacker floods a network or server with an overwhelming amount of traffic to disrupt its normal functioning?

  • Smurf Attack
  • Ping of Death
  • SYN Flood
Correct Answer: SYN Flood

10. What is the term used to describe a cybersecurity incident response exercise conducted to test an organization's preparedness for a real cyberattack?

  • Vulnerability Assessment
  • Penetration Testing
  • Tabletop Exercise
Correct Answer: Tabletop Exercise
