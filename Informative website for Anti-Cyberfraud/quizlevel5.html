Cybersecurity Quiz

1. What is the term used to describe a malicious program that self-replicates and spreads to other computers?

  • Trojan
  • Virus
  • Worm
Correct Answer: Worm

2. Which of the following is NOT a common encryption algorithm?

  • AES
  • DES
  • HTTP
Correct Answer: HTTP

3. What is the term used to describe a cybersecurity attack that involves tricking individuals into revealing sensitive information?

  • Spoofing
  • Phishing
  • Sniffing
Correct Answer: Phishing

4. Which of the following is a type of authentication that relies on physical characteristics such as fingerprints or iris patterns?

  • Two-factor authentication
  • Biometric authentication
  • Token-based authentication
Correct Answer: Biometric authentication

5. What is the term used to describe a software vulnerability that allows an attacker to execute arbitrary code on a targeted system?

  • Buffer Overflow
  • SQL Injection
  • Cross-Site Scripting (XSS)
Correct Answer: Buffer Overflow

6. Which of the following is a secure method for storing and managing passwords?

  • Storing passwords in plain text
  • Using a password manager
  • Sharing passwords via email
Correct Answer: Using a password manager

7. What is the primary purpose of encryption in cybersecurity?

  • To prevent unauthorized access
  • To detect malware
  • To improve network speed
Correct Answer: To prevent unauthorized access

8. What is the term used to describe a security vulnerability that allows an attacker to gain unauthorized access to sensitive data by bypassing normal authentication procedures?

  • Man-in-the-Middle (MitM) Attack
  • Zero-Day Exploit
  • Backdoor
Correct Answer: Zero-Day Exploit

9. What is the term used to describe a cyberattack that aims to trick users into providing sensitive information such as passwords or credit card numbers?

  • Malware
  • Phishing
  • Spoofing
Correct Answer: Phishing

10. What is the term used to describe a security mechanism that prevents unauthorized access to a network by analyzing and filtering incoming and outgoing traffic?

  • Antivirus
  • IDS (Intrusion Detection System)
  • Firewall
Correct Answer: Firewall
